# 蔷薇花园IIROSE：操作与快捷键

> 作者：幽萝Yuro  
> 编辑于 2023年02月28日 18:20

本篇文章用来介绍蔷薇的操作方及快捷键详解。本篇只介绍操作不介绍功能和用法，会有单独文章介绍。

## 通用操作方法

为了提升沉浸感，蔷薇的按钮集中在最上方。

### 取消/后退

- **按钮**：左上角出现 **<** 按钮时，按下即可退回到上个界面。
- **手势/鼠标**：向 **右滑** 动可以取消/后退。
- **键盘**：
    - `Backspace` 键也具有此作用 (需要当前光标不在输入框中)。
    - `Ctrl` + `Backspace` 在光标位于输入框时，可打开主控面板。
- **鼠标侧键**：`Mouse4` (后退键) 也具有此功能，且不受光标位置影响。
- **手机返回**：手机的系统返回键/手势同样有效。

:::tip 小技巧
利用主控面板和聊天界面消息独立显示的特性，可用快捷键一边在主控面板查看历史消息，一边切回聊天界面回复消息。
:::

### 确定/前进

- **按钮**：右上角出现 **☑** 按钮时，按下即可确认并退出。
- **手势/鼠标**：向 **左滑** 动可以确定/前进。
- **键盘**：`Enter` 键也具有此作用 (需要当前光标不在输入框中)。在聊天界面和主控面板时，按下可打开侧边栏。

### 上下滚动

- **键盘**：
    - `Home`: 滑动到顶端。
    - `End`: 滑动到底端。
    - `Space`: 向下滚动一段距离。
    - `↑` / `↓`: 向上/下滚动。
- **鼠标**：
    - **滚轮** 可实现上下滚动。
    - 在概览和聊天界面 **按住左键上下拉动** 可滚动。
    - 按下 **中键** 后移动鼠标可平滑滚动。

### 私聊类

:::warning
注：以下快捷键均在窗口私聊模式无效。
:::

| 快捷键              | 功能                             |
| :------------------ | :------------------------------- |
| `Ctrl`+`Shift`+ `X` | 关闭当前私聊                     |
| `Alt` + `X`         | 在当前私聊界面和群聊界面之间切换 |
| `Alt` + `←` / `→`   | 私聊对象切换                     |
| `Alt` + `↑`         | 切换到群聊                       |
| `Alt` + `↓`         | 切换到最后一个私聊               |
