## 蔷薇简介

蔷薇花园是一个多功能网络聊天室，支持文字聊天，发送音频及点播音乐或视频等各种操作。

在 **搜图神器** 、 **微信小程序** 等小众平台开设宣传入口，用户相对凝聚和稳定，日均在线用户为200人左右。（假期300人左右）

蔷薇花园这个名称有华美而又和谐的含义，希望能给用户带来归属感。主要以ACG相关内容为核心话题，可以边听音乐边聊天或是边看视频边聊天。

网站主以文字聊天为主，但也支持上传音频和视频，除私聊信息储存在本地外不保存房间内聊天信息。

聊天室按照房间为单位，在同一个房间的用户可以进行实时交流，也可以通过“ **弹幕** ”功能进行全站广播，同时支持对单独用户进行私信交流。注册用户可在这里构建自己的个人形象，创建房间及使用媒体点播等全部功能。

网站由站长Ruby和站内用户一同建设和维护，并由用户推举常驻管理员进行环境维护，各个房间亦有 **房主** 和 **星标** 用户管理和维护，站内成员亦共同支持和维护和谐美好的聊天环境，故称为网络世外桃源。

用户私人房间由房主创建和支付站内虚拟货币“ **蔷薇钞** ”进行维持，蔷薇钞可通过在线及聊天来获得，更多信息请看下文教程。

### 房间系统

容纳和分隔用户的容器在这里被称为“房间”，在同一时间内用户只可存在于一个房间中。

<center><img src="https://i0.hdslb.com/bfs/article/4e6973d3bc2dc34037bc9638943c3a33c2fda55b.png" alt="image" referrerpolicy="no-referrer"></center>
<center>蔷薇房间根目录(文件夹)</center>

房间是蔷薇的交流单元（虽然可以私聊但是主以在房间里交流为主），分为官方房和私人房间两大类共四种性质的房间。

其中“社区”即其子目录中的房间是官方创立的公共房间，私人房间由一名“房主”创立并承担相应房租并可设立多名“星标”成员进行管理。房间主要由房主进行装修并设定主题和活动，以供蔷薇居民串门交流。

每有正式的蔷薇居民在沙盒或旅馆类型的房间停留一小时，就会为房主增加一定的系统奖励收入。

| 房间类型 | 房租          | 房主收入  | 备注             |
| :------- | :------------ | :-------- | :--------------- |
| **住宅** | 购买价 30000g | 无        | 永久房间         |
| **旅馆** | 3g/h          | 1g/人/h   | 高活跃性交流场所 |
| **沙盒** | 1g/h          | 0.5g/人/h | 低活跃性交流场所 |

注：只有绑定手机号且绑定了微信/谷歌账户的蔷薇用户才能给房主带来收益，而对房主无限制。

### 蔷薇货币

“ **蔷薇钞** ”（简称： **花钞** /g，原名蔷薇币）是蔷薇花园内的虚拟货币。

在园中的主要作用有:支付房间的开销（租房），购买饰品，购买住宅，更换称号，购买虚拟形象，模拟炒股及购买蔷薇用户私人店铺服务或用于站内悬赏亦或是赠送好友等。

#### 获取方式

注：挂机和聊天收入只有绑定手机号和微信/谷歌帐号后方可获得且不允许用非人手段代劳。

1. 累计在线1小时能自动入账1花钞，俗称挂机。
2. 累计聊天7分钟会自动入账1花钞（每30秒一采样，共采样14次）
3. 经营沙盒/旅馆获得系统奖励
4. 开设站内店铺售卖私人服务
5. 使用花钞进行模拟炒股
6. 找人I你50

### 基础操作

- **头像**：左侧 **导航栏** 的 **导航** 目录下点击“ **编辑** ”或点击个人消息前的头像后选择 **编辑** 后可点击头像替换图片。
- **点歌**：发送`@`➕歌名会自动进入选歌界面，选择歌曲点击 **点播** 出后会进入歌单列表顺序播放。
- **切歌**：发送“ **cut/切** ”可切掉自己点出的歌曲，非己媒体需三个人进行此操作。
- **存图**：长按/右键别人发出的图片，点击“ **加入图包** ”。
- **发图**：点击最左下角的个人头像，在图像栏下会显示自己的图包以及上传个人图片选项。
- **撤回**：长按/右键自己发出的消息并点击 **撤回**。
- **回复**：长按/右键一条信息并点击 **引用**。
- **艾特**：点击对方头像后选择“ **@艾特** ”功能或点击对方消息框后的 **昵称** 即可获取@文本。
- **弹幕**：发送`~`➕空格➕要说的话语可发送“ **弹幕** ”文本在界面右上角弹出，全站用户可见。
- **点赞**：点击对方头像并点击 **查看资料** ，在最下方的“ **赞** ”栏目点击👍图标即可为对方点赞。
- **关注**：点击对方头像并点击 **查看资料** ，在“ **关注** ”这一栏点击进入界面后点击右上角的“ **关注** ”。
- **转账**：点击对方头像并点击 **查看资料** ，在“ **金钱** ”栏目点击💳图标后输入备注和一定金额的花钞即可转账。
- **传送**：点击对方头像并点击 **查看资料** ，在下方“ **坐标** ”这一栏点击文本后点击“ **进入** ”即可加入到同一房间。
- **订阅**：在空白处长按/右键，点击“ **查看房间资料** ”，然后点击“ **订阅** ”栏，点击在右上角的“ **订阅** ”按钮。
- **跟随**：点击别人传送留下的 **足迹** 或 **房间坐标** ，点击“ **进入** ”即可来到目标房间。
- **换房**：在房间界面按住并向左滑动两次进入 **热推房间列表** ，长按/右键后点击“ **进入** ”。
- **回归**：在房间界面按住并向左滑动两次进入 **热推房间列表** ，再下方可显示 **历史房间** 和 **订阅房间**。
- **探索**：在空白处长按/右键，点击“ **目录** ”，然后点击左上角“ **根** ”可进入房间根目录来探索更多房间，点击搜索图标可精准搜索用户和房间。
- **调节音量**：在左侧导航栏的“ **其他** ”栏目下可调节或关闭媒体音量或开关 **媒体加载。**
- **开关壁纸**：在左侧导航栏的“ **其他** ”栏目下可点击“ **壁纸加载** ”来开关房间个性化壁纸 **。**
- **重载**：在左侧导航栏的“ **系统** ”栏目下可点击“ **重载** ”来刷新，以解决部分BUG。
- **私聊**：点击对方头像后点击“ **私聊** ”可进入私聊页（可同自己私聊，私聊消息均储存在本地）
- **群聊**：在私聊状态下按住左滑 **一次** 可进入历史聊天列表，点击第一行的房间即可回归房间群聊。

### 媒体点播

首先媒体点播要在支持点播的房间内进行。

在输入框输入`@`符号，后面跟上歌曲名然后按下回车进入了选歌界面。在上方可点击可选择网易云，QQ音乐，酷狗音乐平台。只能点播免费的音乐资源，付费资源只有试听/无法播放。

点击要点播的歌曲下方的“ **点播** ”字样即可点播而点击封面会跳转到源地址。点播成功会发送媒体信息并在该媒体开始播放时浮出信息卡牌。

点播媒体按照房间歌单的媒体排序来顺序播放，拥有管理权限可在终端调节媒体顺序。

- 发送`cut/切`：可关掉正在播放的自选媒体(cut all 切掉所有，非己点播的媒体需要三人来切)
- 发送`@`➕空格➕歌单名/歌单链接可点播歌单
- 点播视频：发送`#`➕视频名称，然后选择平台和视频进行点播

如果没有看到视频则可 **重载** 或在左侧导航栏里打开“ **其他** ”栏，保证关闭了睡眠模式且打开了媒体加载。打开影院模式可小窗口播放，关闭影院模式可将视频化为背景播放。

#### 自定义点播

当无法使用官方点播或媒体卡顿时可自定义点播，自定义点播需要自行去获取媒体的外链。外链可以通过电脑浏览器F12获取网络媒体资源，手机浏览器Via的资源嗅探，一些网站的资源解析功能获取。

外链具有时效性，但是一般能流畅播放。媒体点播成功且浮出卡片但黑屏时可尝试重载。

## 规范和解释

### 常驻成员

由于网站为站长Ruby一手创建，平时的维护工作就由蔷薇的常驻用户自发分担，故贴入常驻用户表供新人寻求帮助。

**注:**
- 只记录在线时长超过4000小时且活跃的成员，且由在线时长排序，如有不妥可申请修改资料。
- 如有不满要求但想为新人提供帮助者亦可入列。
- 新人如需帮助，可至站内咨询常驻成员。

### 唯一标识（UID）

每个注册用户均有一个唯一且无法更改的UID，发送到界面后可用来点击查询具体成员信息。

点击目标成员头像后点击“ **查看资料** ”后再次点击资料页的头像/昵称后可找到“ **唯一标识** ”选项，点击后会粘贴在输入框中。基本格式为“ **`[@1iloveiirose1@]`** ”由数字和小写字母共13个核心字符外加UID标识外壳且前后均有空格，在蔷薇发送后点击可超链接到具体成员的资料卡。

作用和以“ **`[*昵称*]`** ”为蔷薇@格式的功能相同但不受昵称更改即失效的影响。
