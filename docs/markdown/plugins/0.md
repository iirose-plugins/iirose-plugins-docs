# 插件生态

欢迎来到我们的 IIROSE 插件生态系统！

这里列出的是由我们官方维护的 Koishi 插件，它们旨在与 `adapter-iirose` 无缝集成，为您提供开箱即用的强大功能。

我们致力于不断扩展和完善这个生态，同时也热烈欢迎社区开发者的贡献。

如果您发现了 Bug、有新的功能建议，或者希望直接贡献代码，请不要犹豫，随时通过 GitHub Issues 或 Pull Requests 与我们交流。

## 官方插件列表

### [koishi-plugin-iirose-welcome](https://github.com/iirose-plugins/koishi-plugin-iirose-welcome)

一个入群欢迎插件，当有新用户加入时发送欢迎语。

还能处理用户离开、刷新等情况。

### [koishi-plugin-iirose-follow](https://github.com/iirose-plugins/koishi-plugin-iirose-follow)

让BOT跟随用户切换房间。

### [koishi-plugin-iirose-stock-monitor](https://github.com/iirose-plugins/koishi-plugin-iirose-stock-monitor)

用于监控 IIROSE 股票市场信息，并能在价格变动时进行推送通知。
