# 客户端主动动作

本文档记录了客户端可以主动向服务器发起的各种动作请求。

:::tip
切换房间是一种 **特殊的登录操作**。
:::
## 切换房间

- **目的**: 移动到另一个房间。

### 发送格式

```log
m5fdcb9b634621
```
:::warning
>杺月乌: 2025-11-08 23:16:49

以上内容发完之后 会回一个包

然后再次上线在登陆包里需要携带来的地址（lr字段）才会在原本房间成功下线

不然会出现只上线不下线的奇观

这是iirose的一个bug，可以原地上线（隐式传送）
:::

### 接收格式

发完`m5fdcb9b634621`，随后会收到：
```log
m
```

然后需要上线，**在登陆包里携带旧房间地址**，详情请查看 [登录报文格式](./login.md)

## 获取用户资料

- **目的**: 请求指定用户的详细个人资料。

### 发送格式
`+-` + `用户名称`
```log
+-anata baka？
```
### 接收格式

