服务器事件与通用指令 ​

本文档记录了客户端可以接收到的服务器主动推送事件，以及可以向服务器发送的通用查询指令和其响应格式。

被动接收的服务器事件 ​

这类事件由服务器在特定条件触发时主动推送给客户端。

用户进入房间 ​

  • 触发时机: 当有新用户进入客户端所在的房间时。
  • 对应解码器: JoinRoom.ts

接收格式 ​

log
"1762414198>http://r.iirose.com/i/24/10/11/17/4017-4V.jpg>上学>'1>s>614530>2>>679a51f1d4893>''1867'57'2,100,0.5>>15fdcb9b634621'd''''

用户离开房间 ​

  • 触发时机: 当有用户离开客户端所在的房间时。
  • 对应解码器: LeaveRoom.ts

接收格式 ​

log
"1762414198>http://r.iirose.com/i/24/10/11/17/4017-4V.jpg>上学>'1>s>614530>2>>679a51f1d4893>''1867'57'2,100,0.5>>15fdcb9b634621'd''''

媒体播放更新 ​

  • 触发时机: 房间的媒体播放列表或当前播放状态发生变化时。
  • 对应解码器: Music.ts, MusicMessage.ts

接收格式 ​

log
[此处填写媒体播放更新的示例报文]

主动查询指令与响应 ​

这类交互由客户端主动发起一个简单的指令，服务器则会返回相应的数据。

金融信息 ​

查询股价 ​

  • 请求指令:
    log
    >#
  • 响应格式:
    log
    >1528"717.97753473892"0.46988058556212"0"19.691

查询银行信息 ​

  • 请求指令:
    log
    >*
  • 响应格式:
    log
    >$4823882.474"2987.0952051803"0 0"12 0"19.691"0 0.0123"160

查询余额 ​

  • 请求指令:
    log
    =$
  • 响应格式: (余额为19.691)
    log
    `$19.691

交互式指令 ​

随机骰子 ​

用法

客户端发送 )@ 加上 0 到 7 之间的一个数字，会触发对应数字的官方机器人在公屏回复一个随机数字。

  • 请求指令 (示例):
    log
    )@0
  • 响应格式 (由机器人公屏回复):
    log
    "1762415328>system/901007>艾B> [*上学*]    :   7995>7e7e7e>7e7e7e>2>>5eb9344c0cb93>c伴你左右''0'0'0,0,0.5>690c52e0353e8
